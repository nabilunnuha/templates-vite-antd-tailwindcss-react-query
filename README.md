# Frontend Vite Template

This project is a template for building modern React applications using Vite. It comes pre-configured with essential tools such as TypeScript, Tailwind CSS, React Query, Ant Design, and ESLint.

## Features

- **React 18** for building interactive user interfaces.
- **Vite** for fast and modern development experience.
- **React Router** for client-side routing.
- **Ant Design** for elegant UI components.
- **React Query** for powerful server-state management.
- **Tailwind CSS** for utility-first styling.

## Installation

Before you start, make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1. Clone the repository:
   ```bash
   git clone https://github.com/nabilunnuha/templates-vite-antd-tailwindcss-react-query
   cd templates-vite-antd-tailwindcss-react-query
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run code:
   ```bash
   npm run dev
   ```

### Technologies Used
- [React](https://reactjs.org): A JavaScript library for building user interfaces.
- [Vite](https://vitejs.dev): A fast build tool that serves your code with hot module replacement.
- [Ant Design](https://ant.design): A design system for enterprise-level products.
- [React Query](https://tanstack.com/query/latest): Data-fetching and server-state management for React.
- [React Router](https://reactrouter.com): Declarative routing for React applications.
- [Tailwind CSS](https://tailwindcss.com): A utility-first CSS framework.
